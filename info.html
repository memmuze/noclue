Good to know

person_add

page

  1. When entering the player names, make sure that their order matches the in-game order. This ensures that the assistant works in the correct way.

  2. The amount of assigned cards cannot exeed 18. You must enter unique player names.

  3. Next, you can select the cards that are in your hands. You do not have to assign leftover cards to yourself, you can do it later.

  4. The amount of your selected cards must match the previously set amount.

library_add

page

  1. If there are any leftover cards, you can select those on this page.

  2. The amount of selected leftover cards is calculated by the game engine.

  3. You can record each player's suggestions here. If no one could show a card, select the "No One" option in the "Who showed" menu.

analytics

page

  1. If you wish to remove the last suggestion, click the Undo button at the bottom of the page.

  2. If a clue in the envelope has been found, the page's icon will change color.
